Getting Started

    # To get started, first download and put the GammaJS.js file inside your working directory. 

    # Make sure jquery is installed and if not, run npm i jquery in the terminal. 

    # Then in the html file which you want to use GammaJS, type in both
 
        <script defer type="text/javascript" src='./<YOUR_DIRECTORY>/GammaJS.js'></script> 

        <script defer src= "https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></sscript> 

    # in the head section. 

    # Now you have successfully installed the GammaJS library, start using all its funcionalities in the corresponding javascript file 

    # that modifies the html file which you have just put the scripts into.

Methods

All the global methods available.

To get the methods working, the corresponding generator, which are MusicGenerator, MovieGenerator, GameGenerator, BookGenerator, would have to be
initiated first, as shown in the examples.

IMPORTANT: All parameters(except "popup" for the createNewProfile methods) are required for the methods to work and must be filled in the correct order as listed.

Generator Method Parameter Description
MusicGenerator createNewProfile
  • parent: the DOM element which the new profile would be added to
  • template: the template of the new profile
    • Type: Integer
    • Choose from 1-4, any other number would result in default template of #1
  • musicPic: picture of the music content
    • Type: String
    • '<URL TO THE PICTURE OF THE MUSIC>'
  • musicname: name of the music content
    • Type: JSON
    • {"name": '<NAME OF THE MUSIC>', "link": <'THE LINK TO IT'>}
  • musicartist: artist(s) of the music content
    • Type: Array[JSON]
    • [{"artist": '<ARTIST OF THE MUSIC>', "link": <'THE LINK TO IT'>}]
  • musicgenre: genre(s) of the music content
    • Type: Array[JSON]
    • [{"genre": '<GENRE OF THE MUSIC>', "link": <'THE LINK TO IT'>}]
  • musicrating: rating of the music content
    • Type: Integer
  • musicintroduction: introduction of the music content
    • Type: String
  • reviewLink: link to the review page for the music content
    • Type: String
  • popup: value to determine whether to show the popout window
    • Type: Integer
    • Choose between 0 and 1, 0 means no-popout and 1 means has popout
    • If this field is not filled out by the user, it would be treated as value 0
Create a new profile of a music content with the information provided.
