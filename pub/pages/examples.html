Music Cards And Carousel.

Music cards with 4 different templates and carousel.
Popout window feature:
- Hover on the first card, which has the popout window enabled to see the popout window.
Comparison list feature:
- Click the compare button to see all the music content available for comparison,
- type in the search bar above to search in matching name, artist, genre,
- then click the compare button next to the music content you want to compare to see the comparison table
Comparison drag feature:
- Drag the card on to another card to see the comparison table.
Carousel can be slided using the button on each side or clicking the progress bar at the bottom, comparison drag function is also available.
Carousels would loop itself and slide every 3 seconds automatically, this can be interuptted by hovering over the Carousel

Comparison drag feature:
- Drag the card on to another card to see the comparison table.
Carousel can be slided using the button on each side or clicking the progress bar at the bottom, comparison drag function is also available.
Carousels would loop itself and slide every 3 seconds automatically, this can be interuptted by hovering over the Carousel

Book Cards And Carousel.

Book cards with 4 different templates and carousel.
Popout window feature:
- Hover on the first card, which has the popout window enabled to see the popout window.
Comparison list feature:
- Click the compare button to see all the books available for comparison,
- type in the search bar above to search in matching name, author, genre,
- then click the compare button next to the book you want to compare to see the comparison table
Comparison drag feature:
- Drag the card on to another card to see the comparison table.
Carousel can be slided using the button on each side or clicking the progress bar at the bottom, comparison drag function is also available.
Carousels would loop itself and slide every 3 seconds automatically, this can be interuptted by hovering over the Carousel
